## Quick Start

在使用前你需要首先安装[Offline AI Client](https://npmjs.org/package/@offline-ai/cli).

## Install

```bash
#如果没有,请先安装客户端:
npm install -g @offline-ai/cli
#不需要安装插件,这是默认内置插件
#ai plugin install cmd-brain
```
